# 3D EXPERIMENT BACKUP

Если geo/3D эксперимент окажется неудачным, откат нужно делать не по памяти, а по git/истории чата.

Критичный рабочий файл для отката:
- `src/components/MindMap3D.tsx`

Правило:
- перед любыми крупными 3D-экспериментами сохранять отдельную backup-копию в markdown или git commit;
- не хранить backup как `.tsx`, чтобы не ломать TypeScript build.

Текущий эксперимент:
- резиновая физика сохранена;
- связи строятся через `linkPositionUpdate`, чтобы не отваливались при движении force-графа;
- орбиты вокруг выбранного шара сделаны через `TorusGeometry`.
